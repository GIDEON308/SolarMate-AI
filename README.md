# SolarMate AI

SolarMate AI is an offline AI assistant designed to help solar PV technicians with system sizing, installation, maintenance, and fault diagnosis.

## Features

- Solar PV sizing calculator
- Battery sizing
- Inverter recommendation
- Solar fault diagnosis (coming soon)
- Offline AI knowledge base (coming soon)
